# Moverio AR Workflow

A step-by-step AR demo app designed for the Epson Moverio BT-45C smart glasses. Built using the MAXST AR SDK.

## Features

- AR Marker Detection
- Workflow Step Navigation
- Smart Glasses UI Showcase
